# Bootstrap_Task2

This Project contains a Bootstrap-based landing page for a mobile app marketing template called Appvilla.

The landing page is designed to showcase an app with a hero section, feature highlights, pricing plans, and footer navigation. It is intended for small businesses or startups that want a clean, modern product showcase.

## What it includes

- Hero section with app visuals and call-to-action buttons
- Feature cards to highlight key benefits
- Pricing section with multiple plans
- Footer with company and legal links
- Responsive design using Bootstrap 5

## Files

- `index.html` - Main landing page HTML.
- `css/style.css` - Custom styles for the page.
- `js/script.js` - Custom JavaScript behavior.
- `images/` - Images used by the page.

## How to preview

Open `index.html` directly in a browser, or use a local server for a better experience.

### Local server example

```bash
cd Bootrap_task2
python -m http.server 3000
```

Open `http://127.0.0.1:3000` in your browser.

## Notes

- Bootstrap 5 is loaded via CDN in the HTML head.
- Bootstrap Icons are also loaded via CDN.
- The navbar uses anchor links like `#home`, `#features`, and `#pricing`.
- Custom behavior and active link styling are managed in `js/script.js`.

## Customization

- Edit `css/style.css` to change colors, spacing, and animations.
- Edit `js/script.js` to update section animation or navbar behavior.
